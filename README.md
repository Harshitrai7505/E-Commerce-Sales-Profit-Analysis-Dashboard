# 📊 E-Commerce Sales & Customer Insights Dashboard

## 📌 Project Overview

This project focuses on analyzing an E-commerce dataset using **SQL and Power BI** to derive meaningful business insights. The goal was to understand sales performance, customer behavior, and profitability trends through data cleaning, transformation, and visualization.

---

## 🎯 Objectives

* Analyze overall business performance
* Identify top-performing regions and categories
* Detect loss-making products
* Understand customer purchasing behavior
* Track sales trends over time

---

## 🛠️ Tools & Technologies

* SQL (MySQL) – Data Cleaning & Transformation
* Power BI – Data Visualization & Dashboard
* Excel – Initial data understanding

---

## 🧹 Data Cleaning (SQL)

* Checked and handled NULL values
* Verified duplicate records
* Converted date columns into proper DATE format
* Standardized column names and formats

---

## 🔄 Data Transformation

* Created new columns:

  * Order Year & Month
  * Profit Margin (%)
  * Delivery Days
  * Discount Levels
* Prepared dataset for analysis

---

## 📊 Key KPIs

* Total Sales: **$2.3M**
* Total Profit: **$283K**
* Total Orders: **5K**
* Average Order Value: **$461**
* Profit Margin: **12.45%**

---

## 📈 Dashboard Features

* Sales Trend Analysis (Monthly)
* Sales by Category & Sub-Category
* Region-wise Performance
* Customer Segment Analysis
* Interactive Filters (Region, Category, Year)

---

## 💡 Key Insights

* West region generates the highest sales
* Technology category is the top revenue driver
* Some sub-categories are loss-making despite high sales
* Sales peak during October–November (seasonality)
* Consumer segment contributes the most orders

---

## 📌 Business Recommendations

* Optimize discount strategies to improve profitability
* Focus on high-performing categories like Technology
* Improve performance in underperforming regions
* Reduce losses in low-profit sub-categories
* Leverage peak months for marketing campaigns
